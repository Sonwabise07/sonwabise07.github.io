Academic Project Highlights

DUT Lost and Found Portal (Web App)

The Problem

At a large university like DUT, students frequently lose personal items. The existing methods for reporting and finding them were inefficient, relying on informal social media groups, leading to stress, wasted time, and financial loss for students.

The Solution

As a lead programmer, our team spearheaded the development of a centralized, user-friendly web application using Python. The platform allows students and staff to easily report lost items and search for found ones using detailed descriptions, categories, and images to ensure accurate matching. The system includes secure user authentication and an admin panel for managing the item lifecycle from "lost" to "reclaimed."

The Outcome

The result is a structured and transparent system that significantly reduces the time and frustration involved in recovering lost items. It provides a secure platform that minimizes fraudulent claims and improves overall student satisfaction by providing a reliable campus service.

Key Learnings

This project was a deep dive into full-stack web development with Python. I honed my skills in backend logic, database management, and creating a seamless user experience. It reinforced the importance of clear requirements gathering and iterative development based on user feedback.

Peer Tutoring Platform (Mobile App)

The Problem

Many students, especially first-years or those in challenging courses, need academic support but struggle to find qualified and available tutors. There was no formal system to connect students seeking help with those who could provide it.

The Solution

Our team developed a native Android mobile application using Java. The app creates a student-driven academic support system by connecting tutees with verified peer tutors. Key features include secure user registration, detailed tutor profiles, session booking with MS Teams integration, a rating system, and an AI chatbot for user assistance.

The Outcome

The platform empowers students to get timely, personalized academic help while also allowing tutors to earn an income. The result is a collaborative learning ecosystem that improves academic performance and student engagement within the university community.

Key Learnings

Developing this mobile app in Java was a significant learning curve. It taught me the fundamentals of Android development, working with backend services like Firebase for real-time data and authentication, and integrating third-party APIs like MS Teams and AI chatbots. It was a fantastic experience in building a complex, feature-rich mobile solution from the ground up.

Mr. Onu's Laundromat Booking System

The Problem

A local laundromat was using a manual, paper-based system for booking services, which was inefficient, prone to errors, and offered no convenient way for customers to pay or track their service status.

The Solution

Our team created a desktop application using C#. The system allows users to book services, process online payments, and receive notifications. We designed the entire system architecture, including WBS, Use Case, and ERD diagrams, and engineered the backend SQL database to manage all customer and transaction data.

The Outcome

The final product was a successful prototype that streamlined the laundromat's entire booking and payment process. It demonstrated a viable platform for modernizing their operations, improving customer experience, and increasing efficiency.

Key Learnings
